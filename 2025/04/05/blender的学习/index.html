Lonley2025-04-052025-04-05

学习视频是【【Kurt】Blender零基础入门教程 | Blender中文区新手必刷教程(已完结)】

一、视图操作

  • 数字键：

    • 数字键 1（主键盘）：切换到前视图。

    • 数字键 3（主键盘）：切换到侧视图。

    • 数字键 7（主键盘）：切换到顶视图。

    • 数字键 5（主键盘）：在正交视图和透视视图之间切换。

  • 鼠标操作：

    • 鼠标滚轮：放大或缩小视图。

    • 按住鼠标中键并拖动：平移视图。

    • 按住鼠标中键并按住Shift键拖动：旋转视图。

  • 其他快捷键：

    • Numpad .：聚焦到选中的对象。

    • Shift + C：重置视图到默认的场景中心。

二、选择操作

  • 单个对象选择：

    • 鼠标左键：在3D视图中选择对象。

    • Shift + 鼠标左键：添加或移除对象的选择。

  • 多个对象选择：

    • Ctrl + 鼠标左键：框选多个对象。

    • B：框选工具，用鼠标拖动一个框来选择框内的对象。

    • C：圈选工具，用鼠标画圈来选择圈内的对象。

  • 其他选择：

    • A：全选或取消全选。

    • Alt + 鼠标左键：选择边缘环（在编辑模式下，用于选择边缘环）。

三、物体模式操作

  • 移动、旋转和缩放：

    • G：移动物体。

    • R：旋转物体。

    • S：缩放物体。

    • 在执行移动、旋转或缩放操作后，可以按X、Y或Z键来限制操作到特定的轴。

  • 复制和删除：

    • Shift + D：复制选中的物体。

    • X：删除选中的物体。

  • 其他操作：

    • Tab：在物体模式和编辑模式之间切换。

    • H：隐藏选中的物体。

    • Alt + H：显示所有隐藏的物体。

四、编辑模式操作（以网格编辑为例）

  • 顶点、边和面的切换：

    • 1（数字键区）：切换到顶点选择模式。

    • 2（数字键区）：切换到边选择模式。

    • 3（数字键区）：切换到面选择模式。

  • 创建和删除：

    • F：创建新的面或边。

    • X：删除选中的顶点、边或面。

  • 其他操作：

    • E：挤出选中的顶点、边或面。

    • Ctrl + R：添加循环边。

    • Ctrl + B：倒角操作。

五、渲染和动画

  • 渲染：

    • F12：渲染当前场景。

    • Shift + F12：渲染并保存当前场景。

  • 动画：

    • I：插入关键帧。

    • Alt + A：在3D视图中播放动画。

六、其他常用快捷键

  • Ctrl + Z：撤销操作。

  • Ctrl + Shift + Z：重做操作。

  • Ctrl + S：保存文件。

  • Ctrl + O：打开文件。

  • Ctrl + N：重置场景。
